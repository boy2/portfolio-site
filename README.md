# ポートフォリオサイト

React、Vite、Vikeで構築されたモダンなポートフォリオウェブサイトです。

## はじめに

依存関係のインストール:

```bash
npm install
```

開発サーバーの起動:

```bash
npm run dev
```

ブラウザで、コンソールに表示されたURLを開いて結果を確認してください（通常は `http://localhost:3000` ですが、ポートが使用中の場合は別のポート番号が割り当てられます）。

## ビルド

本番環境用のビルド:

```bash
npm run build
```

本番ビルドのプレビュー:

```bash
npm run preview
```

## 技術スタック

- [React](https://react.dev/) - UIライブラリ
- [Vite](https://vite.dev/) - ビルドツール
- [Vike](https://vike.dev/) - Reactフレームワーク
- [Tailwind CSS](https://tailwindcss.com/) - CSSフレームワーク
- [TypeScript](https://www.typescriptlang.org/) - 型安全性
